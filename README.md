## ✨ 特性

- 💪 使用 TypeScript 最新特性开发
- 🐆 全面基于 unbuild 进行打包构建
- 🛠 功能强大，满足日常开发的大部分场景
- ✊ 增加支持链表、队列等数据结构
- 🪐 不同打包模式，兼容不同项目
- 🏆 支持完整引入和按需引入
- 💨 配置简单，上手容易
- 🚩 使用 vitest 单元测试
- 👍 社区团队维护
- 📃 优质详细的文档
